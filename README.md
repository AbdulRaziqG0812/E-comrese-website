# 🛍️ E-Commerce Website

A fully functional E-Commerce web application built with **Python Flask**, **MySQL**, **HTML**, **CSS**, and **JavaScript**. The project provides a complete online shopping experience, including user authentication, product browsing, shopping cart, checkout, order management, and an admin dashboard.

---

- User Registration & Login
- Browse Products

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/AbdulRaziqG0812/E-comrese-website.git
```

### 2. Navigate to Project Folder

```bash
cd E-comrese-website
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure Database

- Create a MySQL database.
- Import the SQL file from the **sql** folder.
- Update your database credentials inside `store.py`.

### 5. Run the Application

```bash
python store.py
```

Open your browser and visit:

```
http://127.0.0.1:5000/
```
